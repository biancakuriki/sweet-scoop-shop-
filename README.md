# Sweet Scoop Shop

Sweet Scoop Shop is a cute browser game where you serve animal customers by building the right ice cream order before time runs out. You can earn score, collect dollars, unlock new flavors and cups, and keep progressing with saved shop upgrades.

Live game on GitHub Pages:

https://biancakuriki.github.io/sweet-scoop-shop-/

## How To Play

- Press `Play` from the shop screen.
- Read the customer's order.
- Build the ice cream with the correct container, flavor, and topping.
- Press `Serve` to submit the order.
- Earn score and collect dollars as you play.
- Spend dollars to unlock more menu items in the shop.

## Features

- 60-second arcade-style gameplay
- Animal customers with different orders
- Bunny bonus customer and phoenix shop event
- Unlockable flavors and containers
- Local save progress using browser storage
- Leaderboard display on the start screen

## Project Files

- `index.html` - game layout and screens
- `styles.css` - visual design and responsive styling
- `script.js` - game logic, shop system, saving, and events

## Run Locally

Because this is a static web project, you can open `index.html` directly in a browser or serve the folder with a simple local server.

Example:

```sh
python3 -m http.server
```

Then open `http://localhost:8000`.
